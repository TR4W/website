FKEY HEIRARCHY (CommonMessages.ini being highest priority)

  • If key exists in CM it will be active else
  • If key exists in contest.cfg or tr4w.ini it will be active else
  • If key exists in contest specific defaults it will be active else
  • If key exists in global defaults it will be active
