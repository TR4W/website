The following may be imbedded in CW messages:


# QSO number

% database name

~ GM/GA/GE

: Enable keyboard CW

^ half space

] repeat RST sent

@ Call window contents

$ GM + name

| received name

\ My callsign

'^F WPM+2 (hold down CTRL then press 'P' then (with control still down) press 'F'

^S WPM-2 (" 'S' "

+ AR

< SK

= BT

! SN

& AS

) last QSO''s call

To program control characters, press Control-P first then control character.

To program your F keys,

Example: CQ F Key (ALT-P) and then 'C' -- ' CQ^TEST \ \ '

or

Exchange F Key (ALT-P) and then 'E' - '5NN #'

hint: always use upper case characters--

Example 'tu \' WRONG

'TU \' OK
